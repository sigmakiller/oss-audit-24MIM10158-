## Project Description

This project is an audit of the open-source software Git. It explores its origin, licensing model, ethical aspects, Linux implementation, and ecosystem. The project also includes five shell scripts demonstrating practical Linux and scripting skills.

## Repository Contents

* script1.sh → System Identity Report
* script2.sh → FOSS Package Inspector
* script3.sh → Disk and Permission Auditor
* script4.sh → Log File Analyzer
* script5.sh → Open Source Manifesto Generator
